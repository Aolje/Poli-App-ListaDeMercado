# 🛒 App de Lista de Compras Familiar

## ✨ Propósito

En muchos hogares es común olvidar productos esenciales al momento de hacer mercado, lo cual genera desorganización y pérdida de tiempo. Además, la falta de comunicación efectiva entre los miembros de la familia puede empeorar este problema.

Este proyecto nace con el objetivo de **organizar las compras del hogar** de forma colaborativa, permitiendo que todos los miembros de la familia (padre, madre, hijos, abuelos, etc.) mantengan una lista de compras **sincronizada en tiempo real**, mejorando la planificación y evitando olvidos.

---

## 🧩 Tecnologías utilizadas

### 🖥️ Frontend

- [React](https://reactjs.org/)
- [Firebase](https://firebase.google.com/) (Autenticación, Firestore y Hosting)
- [Vite](https://vitejs.dev/) (opcional, si usas Vite en lugar de Create React App)
- [TailwindCSS](https://tailwindcss.com/) (opcional para estilos)

### ⚙️ Backend

- Java 17+
- Spring Boot
- Base de datos relacional (PostgreSQL, MySQL, etc.)
- API REST para gestión de usuarios, roles y sincronización

---

## 🚀 Funcionalidades principales

- ✅ Autenticación con Firebase
- ✅ Lista de compras compartida entre usuarios
- ✅ Gestión de productos por categorías
- ✅ Roles personalizados (padre, madre, hijo, abuelo, etc.)
- ✅ Notificaciones en tiempo real (por definir)
- ✅ Integración con backend Java para lógica de negocio adicional
